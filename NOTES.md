## Inspection -> workorder
when an inspection is done, if anything was marked bad immediatly the user must make a workorder, it takes them to that screen. It autofills the problem and description with what they already entred and described, it also fills the date and vehicle id. They can make any edits. They must select a severity, add any photos, then they can submit. This process will repeat for each thing they marked bad on the inspection.
